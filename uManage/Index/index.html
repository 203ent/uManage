Saturday, July 26th, 2014

You have successfully installed uManage and are almost ready to start letting users access the system. There are a few more setup tasks to complete but they are all fairly painless and straightforward. At this point, you have setup the application, connected to the database, and connected to the directory.

  1. Configure Administrative Users & Roles
  2. Configure Locations

Feature Requests

We obviously want to improve the system and make it easier for everyone to use. If you have, a specific feature or ideas in mind feel free to visit GitHub and suggest the feature. We cannot guarantee we will implement everything but we will at least look at your request.

If you want to code something out, feel free to fork/patch the repository and send it in!

Getting Support & Help

uManage is an open source product and as such you can usually get support and additional documentation from GitHub. If you need more assistance or want to inquire about specific customization commercial assistance is provided by 203 Enterprises along with consulting and technical assistance services.
